# journal-zho-yuyan

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容以中文为主要语言，涵盖不同主题和用途。仓库本身不针对任何单一域名或特定网站，仅作为静态页面的集合与发布点。

## 项目简介

这是一个以 HTML 页面为基本单元的归档仓库。每个页面独立存在，彼此之间没有强依赖关系。仓库的目标是提供一个中立的存储与分发空间，方便用户浏览、引用或存档。

## 目录说明

```
.
├── index.html          # 仓库入口页，列出所有归档页面链接
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源目录（可选）
    ├── css/
    └── images/
```

- **index.html**：仓库首页，提供页面索引和导航。
- **pages/**：所有独立 HTML 页面均放置于此目录下，命名方式为 `page-xxx.html`。
- **assets/**：存放页面可能引用的公共样式、图片等资源。

## 页面归档说明

- 每个页面均为完整的 HTML 文件，可独立打开浏览。
- 页面之间不存在跨域或跨站点引用，所有资源尽量本地化。
- 归档内容不涉及具体商业、政治或敏感话题，仅作为资料存储。
- 页面版本通过 Git 提交记录进行管理，历史版本可在仓库提交历史中查看。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或对现有页面进行内容修正。
- 新增页面请遵循以下规范：
  - 页面文件放置于 `pages/` 目录下。
  - 在 `index.html` 中添加对应链接。
  - 页面内不包含恶意代码、跟踪脚本或外部资源引用（需本地化）。
- 如发现页面无法正常显示或存在错误，请通过 Issue 反馈。
